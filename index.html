Projects

Self Driving Car Using Deep Learning on Quanser Qcar

In the last decades, autonomous navigation is a hot topic in the robotics industry where an autonomous agent is able to avoid obstacles and navigate around the map. This requires proper path planning and an obstacle avoidance algorithm. For collision-free autonomous navigation, a robot must be able to detect obstacles and efficiently avoid them. Neural Network architecture inspired by mimicking the human brains proposed in this project by training the weights and storing them in the form of neurons. Implemented Deep Convolutional Neural Network for the proposed navigation system. The proposed model efficiently detect the static and dynamic obstacle in its path and avoid them. It is built in python 3.0 and experimental tests based on a small-scale model Qcar from Quanser state-of-the-art product equipped with Nvidia Jetson TX2 have been conducted to verify the effectiveness of the proposed network.

Clearpath Jackal Frontier Exploration SLAM

Implemented Frontier-based exploration using LiDAR in ROS-Melodic Gazbo for simulated environemnt for autonomous mapping of unknown environments. The autonomous mapping process involves localisation, mapping, and exploration. With the emergence of unmanned aerial vehicles, there is now a need for autonomous exploration algorithms that work in tandem with simultaneous localisation and mapping (SLAM).

Space Invaders Game using Deep Reinforcement Learning

Implement a deep learning model to successfully learn control policies directly from high-dimensional sensory inputs using reinforcement learning. We chose to work with RAM Monitor. In this case, environmental monitoring is provided by the state of the RAM, that is, a set of 128 values. Each value is a byte (an integer from 0 to 255). RAM is a compact representation of state compared to algorithms that display the image as a state. In this environment The reward in the Space Invaders gym environment is the result in the given time step.
